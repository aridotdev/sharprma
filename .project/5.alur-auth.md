# AUTH FLOW DECISIONS (FINALIZED)

## FINAL SPECIFICATIONS 📋
1. User Registration
  - Admin only - Tidak ada self registration
  - Admin creates user melalui admin panel
  - Initial password: sharp1234 (template default)

2. Password Management
  - Force change on FIRST LOGIN only
  - After first login, user can change anytime (optional via profile menu)
  - Password policy: Min 8 char, 1 uppercase, 1 number

3. Session Management
  - Session expires: 7 days
  - No "Remember Me" feature
  - Auto-refresh session during active use

4. Role System
  - Single role per user
  - Roles: ADMIN, QRCC, MANAGEMENT, CS
  - Role tidak bisa berubah (kalau mutasi = create new user)

5. Branch Assignment
  - CS only - tied to specific branch (auto-fill di claim)
  - QRCC, ADMIN, MANAGEMENT - no branch assignment

6. Security Features
  - Failed login: Lock account for 15 minutes after 5 failed attempts
  - Password reset: Admin can reset user password to sharp1234
  - No email notifications (keep simple)
